This directory and its subdirectories contain documentation about
various aspects of the Philanthropy Data Commons framework, data
entities, permissions system, etc.

Much of this documentation is automatically published to the
[PhilanthropyDataCommons.org](https://www.PhilanthropyDataCommons.org)
web site (and is automatically converted from Markdown to HTML along
the way). See [DOCS_AUTOPUBLICATION.md](DOCS_AUTOPUBLICATION.md) for
technical details about this process.
